## Framework prompt áp dụng

- Thứ tự module: Core subject -> Identity anchors -> Primary emotion -> Primary action/pose -> Outfit -> Accessories/props -> Composition and background constraints -> Style block -> Negative prompt
- Nguyên tắc tiết giảm: mỗi prompt chỉ giữ 1 cảm xúc chính, 1 hành động chính, tối đa 1 bộ trang phục chính, tối đa 2 phụ kiện chính
- Nguyên tắc đa dạng: giữ cố định identity anchors, chỉ xoay vòng emotion, action, outfit và props theo mức độ dữ liệu brief
