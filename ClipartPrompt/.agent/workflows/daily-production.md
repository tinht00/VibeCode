# Daily Production Workflow

Quy trình làm việc hàng ngày để duy trì tiến độ dự án Project Lumina.

## I. Time Blocks (Khung Giờ)

### 🌤️ Morning Session (Sáng Tạo - 2h)
- **Mục tiêu:** Tạo ra nguyên liệu thô (Raw Assets).
- **Công cụ:** Dreamina, Midjourney (nếu có).
- **Hoạt động:**
  1.  Chọn 1 Nhân vật (e.g., Poki) và 1 Theme (e.g., Eating, Sleeping, Playing).
  2.  Sử dụng `master-style` prompt từ `skills/lumina-prompting.md`.
  3.  Generate 20-30 hình ảnh raw.
  4.  Lọc lấy 5-10 hình xuất sắc nhất.

### ☀️ Afternoon Session (Kỹ Thuật - 2h)
- **Mục tiêu:** Xử lý hậu kỳ (Post-processing).
- **Công cụ:** Python Scripts, Photoshop/Canva.
- **Hoạt động:**
  1.  Chạy script xóa nền (Background Removal).
  2.  Kiểm tra viền ảnh (Edge Check).
  3.  Upscale lên 4K resolution.
  4.  Lưu vào folder `Assets/Processed`.

### 🌙 Evening Session (Kinh Doanh - 1h)
- **Mục tiêu:** Đóng gói và Phân phối.
- **Công cụ:** Etsy, Pinterest, YouTube Shorts.
- **Hoạt động:**
  1.  Tạo Listing Image trên Canva (Mockup).
  2.  Đăng lên Etsy với SEO keywords (từ `plan.md`).
  3.  Tạo 1 Video Short kể chuyện ngắn về hình ảnh vừa tạo.
  4.  Pin lên Pinterest board.

## II. Checklist Hàng Ngày

- [ ]  Đã tạo ít nhất 1 bộ Clipart mới (5-10 items)?
- [ ]  Đã xử lý sạch nền và upscale chưa?
- [ ]  Đã post lên Etsy chưa?
- [ ]  Đã có 1 Video Short/Pin mới chưa?

## III. Weekly Review (Cuối Tuần)

- **Thứ 7:** Review doanh số Etsy, traffic YouTube. Điều chỉnh hướng content nếu cần.
- **Chủ Nhật:** Nghỉ ngơi hoặc Brainstorm cốt truyện mới cho tuần sau.
