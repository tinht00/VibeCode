# Domain Glossary

## Mục tiêu

Ghi lại thuật ngữ nghiệp vụ và kỹ thuật dùng lặp lại trong workspace để tránh phải giải thích lại qua nhiều thread.

## Thuật ngữ hiện có

- `Workspace VibeCode`: tập nhiều dự án con dùng chung rules và workflow trong `.agent/`.
- `Digest docs`: các file trong `ai-knowledge/` dùng để tóm tắt bối cảnh cho AI, không thay thế code.
- `NotebookLM manifest`: danh sách file docs nên upload vào NotebookLM theo từng nhóm mục đích.
- `Governance`: nhóm tài liệu rules, workflows, skills và project settings.
- `Architecture & Docs`: nhóm README, setup docs, docs kiến trúc và digest docs.

## Cần cập nhật thêm

- Thuật ngữ nghiệp vụ riêng của từng dự án con.
- Tên gọi thống nhất cho các pipeline, batch flow, sync flow hoặc clipart workflow đang dùng thật sự.
