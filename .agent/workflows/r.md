# /r - Đọc lại Rules (Tuân thủ quy chuẩn)

Nhắc AI đọc lại toàn bộ quy tắc dự án trước khi tiếp tục hành động.

## Quy trình
// turbo
1. Đọc file `.agent/rules.md` - Quy chuẩn kỹ thuật và quy trình làm việc.
2. Đọc file `.agent/project_settings.md` - Hướng dẫn vận hành agent.
3. Xác nhận đã hiểu và áp dụng đúng:
   - **Ngôn ngữ**: Trả lời Tiếng Việt, giữ thuật ngữ Anh chuyên ngành.
   - **Tech Stack**: Tuân thủ quy tắc theo công nghệ đang dùng.
   - **Cấu trúc phản hồi**: Phân tích → Code → Lưu ý.
   - **Zero-G Workflow**: Analysis → Implementation → Review.
