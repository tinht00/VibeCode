# Etsy Research MVP - Rules

## Stack

- Backend: Python 3.12+ / FastAPI / SQLAlchemy / Alembic
- Frontend: Vue 3 / Vite / Pinia / Vue Router

## Quy tắc chính

- Tất cả comment và docs viết bằng tiếng Việt có dấu.
- Vue phải dùng `<script setup>` và Pinia setup store.
- Backend phải có type hints cho mọi hàm.
- API trả lỗi bằng tiếng Việt rõ ràng.
- Không thêm logic scraping vào lõi app.
